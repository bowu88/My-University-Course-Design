#FjnuMap--福师大校园地图
##简介
'3S'综合实习时的项目，这也是大学期间个人**最喜欢的作品**。从坐标的测量，到遥感图像校正，再到系统开发，这真的是一个非常有意义的体验。

只要用的的工具有：

* 开源GIS服务器`GeoServer`，用于提供地理数据服务。
* 开源地图前端框架`Openlayers 3`
* twitter的响应式前端框架`Bootstrap`
